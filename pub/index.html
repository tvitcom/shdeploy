shdeploy diagram

After settings on developer host you will run 

sh deploy-server.sh
or use local file-server for deployment.

For start deployment on installed Debian system you will run: 

wget 192.168.10.100:3000/deb9/install.sh && sh install.sh

i. For example we use ip: 192.168.10.100. You should use own appropriate ip address of deploy-host (i.e. fileserver for deployment scripts and configs).
ii. Html and JS code from https://github.com/antirek/network-diagram
